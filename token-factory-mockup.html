Architecture: How It Works

User clicks "Explain Like I'm 5" → Browser calls AWS Lambda Function URL → Lambda sends prompt to Nebius Token Factory API (DeepSeek R1) → Response streams back to the page. API key stays secure in Lambda — never exposed to the browser.
